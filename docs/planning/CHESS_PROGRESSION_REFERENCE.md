# Chess Progression Reference

## Purpose

This file defines the current thematic content for the savings-based chess progression system.

The progression model should use:

- chess piece tiers
- Roman numeral sublevels
- one short subtitle per sublevel
- one short quote with attribution per sublevel

## Pawn

- `I` `Isolated Pawn`
  - "The isolated pawn is a weakness — but also a source of dynamic play."
  - Garry Kasparov
- `II` `Doubled Pawn`
  - "Doubled pawns are often weak, but they may open important files."
  - Siegbert Tarrasch
- `III` `Connected Pawn`
  - "Connected pawns support each other like soldiers in formation."
  - Aron Nimzowitsch
- `IV` `Passed Pawn`
  - "A passed pawn must be pushed."
  - Aron Nimzowitsch
- `V` `Pawn on the Sixth Rank`
  - "A pawn on the sixth rank is stronger than a knight."
  - Aaron Nimzowitsch
